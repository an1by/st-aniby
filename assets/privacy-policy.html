Политика конфиденциальности

1. Сбор и использование данных

Наш сервис минимально обрабатывает данные пользователей для обеспечения работоспособности и безопасности:

  • IP-адреса: Мы временно сохраняем IP-адреса исключительно для регулирования лимитов запросов (не более 100 запросов за 15 минут). Эти данные хранятся в памяти и автоматически удаляются по истечении срока действия ограничений.
  • Access Token: Токены доступа передаются через параметры запроса для взаимодействия с API Donation Alerts, но не сохраняются на наших серверах.

2. Передача данных третьим лицам

Мы не передаём собранные данные (включая IP-адреса) третьим лицам, за исключением:

  • Сервиса Donation Alerts, чьё API используется для обработки запросов. Их политика конфиденциальности регулирует использование данных на их платформе.

3. Безопасность

Мы принимаем меры для защиты данных:

  • Используем стандартные механизмы ограничения запросов (rate limiting) для предотвращения злоупотреблений.
  • Не храним персональные данные пользователей (логины, пароли, платежные реквизиты).
  • Публикуем актуальный исходный код сервиса в публичном репозитории an1by/st-aniby.

4. Права пользователей

Вы имеете право:

  • Поменять ключ пользователя через панель управления Donation Alerts, сбросив и все остальные токены доступа.
  • Запросить информацию о временно сохранённых IP-адресах (если они не были автоматически удалены).

5. Логирование и ошибки

Сервис не ведёт постоянное логирование запросов или ошибок. Исключения обрабатываются в режиме реального времени без сохранения персональных данных.

6. Изменения в политике

Любые изменения в политике конфиденциальности будут опубликованы на этой странице.
